# Pycaline - a simple command-line calendar in Python

## Setup
```
pip install pycaline
```

## Usage
### Add an event
```
pycaline add 14:00 15:00 Meeting
```

### See schedule
```
pycaline schedule
```

```
pycaline schedule tomorrow
```

```
pycaline schedule 2016-04-16
```

### Remove an event
```
pycaline schedule
pycaline remove 2
```
